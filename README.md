# Titanic Survival Data Analysis Dashboard
# Objective :
 To design an interactive Titanic Survival Dashboard in Power BI Desktop that visualize and explore which passengers survived and which did not, and how survival is influenced by variables such as Passenger class (Pclass), sex, Age, Fare, Emdarked, and family size.
 ## Dashboard Design & Insights :
 Structured the report into multiple pages such as :
 Overview page - KPI cards for Total passengers, survivors, Deaths, and survival Rate.
 Survival by class & sex page - Bar and stacked bar charts showing survival by pclass and sex.
 Age & Fare Analysis page - Histograms and box plots for Age and Fare separated by survival status.
 Family & Embarkation page - Survival by Family size, IsAlone, and Embarked.
